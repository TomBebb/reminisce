Reminisce is a lightweight library intended to be used for detecting and reading from joysticks at a low level across many different operating systems.

Scanning for joysticks

To scan for joysticks, a Backend must be created.

Scanning for events

To scan events, a Backend must be created and polled.

use reminisce::{Backend, Native};
let mut backend = Native::new();
for event in backend.iter() {
    println!("{:?}", event);
}
